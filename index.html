Music Composition Assistant

In Development

Problem: AI music generation is largely unhelpful for serious composers, but with the right training and instructions there's a way for AI to become a genuinely useful part of the creation process.

Solution: An agent that doesn't generate music, but rather assists in each phase of the writing process:

  • Research and planning
  • Proposing ideas and analysis from small or incomplete musical fragments (rhythms, melody, harmony)
  • Analyzing works in progress and functioning as a brainstorm partner
  • Assisting in the engraving process as an editor

Goal: Seamlessly integrate into the writing process in a way that genuinely helps, without detracting from or slowing down the creative work.

Current phase: Prompting across each stage of the composition process, grading responses against a rubric, and using the patterns that emerge to train the agent toward better synthesis and analysis.

How I Work with Claude

Problem: Early on, Claude would hallucinate, lose context between conversations, and make changes that broke unrelated parts of the codebase.

Solution: Built structured MD files covering project instructions, backend architecture docs, and workflow protocols to give Claude consistent context so every prompt could be short and focused. Then integrated Claude Code directly into VS Code, which eliminated nearly all hallucinations.

Result: A workflow where I can move fast without breaking things, with tight prompts, live diffs, and a clear audit trail before any change is accepted.

Next step: Building custom Claude Code skills to automate repetitive workflow steps and further refine my development process.
